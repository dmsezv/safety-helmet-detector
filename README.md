# Safety Helmet Detector

## 1. Установка и настройка окружения

> Проект использует uv для управления зависимостями, но совместим со стандартным pip.

### Шаг 1: Клонирование репозитория

```bash
git clone <repo_url>
cd safety-helmet-detector
```

### Шаг 2: Создание виртуального окружения

```bash
# Создание виртуального окружения
python3 -m venv .venv

# Активация (macOS/Linux)
source .venv/bin/activate

# Активация (Windows)
# .venv\Scripts\activate
```

### Шаг 3: Установка зависимостей

Установите проект в режиме редактирования с зависимостями для разработки:

```bash
pip install -e ".[dev]"
```

Для автоматической проверки качества кода перед коммитом:

```bash
pre-commit install
```
